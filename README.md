## Project Overview

The Food Order Website is a responsive web application that allows users to browse food items, select their favorite dishes, and calculate the total order amount. This project focuses on creating an interactive and user-friendly food ordering experience using front-end technologies.

## Features

Browse different food items from the menu

Add multiple items to the order

Automatic total price calculation

Simple and clean user interface

Responsive design for desktop and mobile

Technologies Used

HTML

CSS

JavaScript

clone the repository

Double-click index.html
